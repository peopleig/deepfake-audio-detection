To start training from scratch and save the checkpoints - 
python -m src.train --data_dir ".\data\for-norm" --fixed_seconds 2.0 --augment --pretrained --epochs 20 --batch_size 16 --out ".\runs\for_resnet34.pt"

To resume training from a saved checkpoint - 
python -m src.train --data_dir ".\data\for-norm" --fixed_seconds 2.0 --augment --pretrained --epochs 20 --batch_size 16 --out ".\runs\for_resnet34.pt" --resume ".\runs\for_resnet34.pt"

To evaluate the trained model - 
python -m src.evaluate --data_dir ".\data\for-norm" --ckpt ".\runs\for_resnet34.pt" --fixed_seconds 2.0 --batch_size 32 --threshold 0.5
